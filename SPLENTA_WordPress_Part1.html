SPLENTA

瞬态合成器 / Transient Synthesizer

Version 1.0.0 | VST3 | macOS & Windows

功能

瞬态检测

实时分析信号瞬态特征，阈值、上限、释放时间和频段滤波可调。

双层合成

音高包络与振幅包络独立控制。4种波形、饱和度、白噪声混合。

MIDI支持

虚拟键盘和外部控制器，可作为MIDI乐器演奏。

5种主题

Bronze / Silver / Gold / Platinum / Diamond，实时包络显示。

A/B对比

左键A→B，右键B→A，3D旋转动画。

音频处理

闪避防过载，Split分离模式，15个预设。

应用场景

  • 鼓组增强：添加打击感和力度
  • 低音强化：底鼓和贝斯补充sub频率
  • 瞬态塑形：调整Attack和Decay曲线
  • 声音设计：合成打击乐和特效音色
